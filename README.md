# BotrollNLP-telegram

Simple y pequeño Bot de Telegram para grupos el cual puede crear situaciones como esta:

[![Sin-t-tulo.png](https://i.postimg.cc/MKny1vTW/Sin-t-tulo.png)](https://postimg.cc/vcdgs8fj)


## Funcionalidades

* Scraping criptomonedas y vacunas covid-19
* Detección de faltas ortográficas
* Respuestas en función de localizaciones y/o entidades
* Reconocimiento de URLs no famosas
* Conversación basada en word2vec
